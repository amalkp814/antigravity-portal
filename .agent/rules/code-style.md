# specialized Code Style Rules

> **Note**: These rules are automatically ingested by the Antigravity Agent.

## Python Style (PEP 8)
*   Make sure all the code is styled with PEP 8 style guide.
*   Make sure all the code is properly commented with docstrings.

## JavaScript/TypeScript
*   Use `const` and `let`, avoid `var`.
*   Prefer arrow functions for callbacks.
*   Use strict type checking in TypeScript.
